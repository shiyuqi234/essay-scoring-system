# H2 数据库查看指南

## 方法一：使用 H2 Web 控制台（最简单）

### 步骤 1：启动应用

确保应用正在运行：
```bash
mvn spring-boot:run
```

### 步骤 2：访问 H2 控制台

在浏览器中打开：
```
http://localhost:8080/h2-console
```

### 步骤 3：填写连接信息

在 H2 控制台页面中，填写以下信息：

| 字段 | 值 |
|------|-----|
| **JDBC URL** | `jdbc:h2:file:./data/essayscoring` |
| **用户名** | `sa` |
| **密码** | （留空，不填） |

**注意**：JDBC URL 中的路径是相对于应用启动目录的。如果应用在 `E:\JavaCode\Essay Scoring` 目录启动，则数据库文件位于 `E:\JavaCode\Essay Scoring\data\essayscoring.mv.db`

### 步骤 4：连接并查看数据

点击 **"连接"** 按钮，即可进入数据库管理界面。

### 常用 SQL 查询示例

#### 查看所有表
```sql
SHOW TABLES;
```

#### 查看用户表数据
```sql
SELECT * FROM users;
```

#### 查看作文表数据
```sql
SELECT * FROM essays;
```

#### 查看评分表数据
```sql
SELECT * FROM scores;
```

#### 查看反馈表数据
```sql
SELECT * FROM feedbacks;
```

#### 查看评分规则表数据
```sql
SELECT * FROM scoring_rules;
```

#### 查看表结构
```sql
-- 查看 users 表结构
DESC users;

-- 或者
SHOW COLUMNS FROM users;
```

#### 统计查询示例
```sql
-- 统计用户数量
SELECT COUNT(*) FROM users;

-- 统计作文数量
SELECT COUNT(*) FROM essays;

-- 查看每个学生的作文数量
SELECT student_id, student_name, COUNT(*) as essay_count 
FROM essays 
GROUP BY student_id, student_name;

-- 查看平均分
SELECT AVG(total_score) as avg_score FROM scores;
```

## 方法二：使用数据库管理工具

### 使用 DBeaver（免费，推荐）

1. **下载 DBeaver**
   - 访问：https://dbeaver.io/download/
   - 下载并安装

2. **创建新连接**
   - 打开 DBeaver
   - 点击 "新建连接" → 选择 "H2"
   - 填写连接信息：
     - **路径**: `E:\JavaCode\Essay Scoring\data\essayscoring`
     - **用户名**: `sa`
     - **密码**: （留空）

3. **连接数据库**
   - 点击 "测试连接"
   - 如果成功，点击 "完成"

### 使用 IntelliJ IDEA Database 工具

1. **打开 Database 工具窗口**
   - 在 IDEA 中，点击右侧的 "Database" 标签
   - 或通过菜单：View → Tool Windows → Database

2. **添加数据源**
   - 点击 "+" → Data Source → H2
   - 填写连接信息：
     - **URL**: `jdbc:h2:file:E:/JavaCode/Essay Scoring/data/essayscoring`
     - **User**: `sa`
     - **Password**: （留空）

3. **测试并连接**
   - 点击 "Test Connection"
   - 如果成功，点击 "OK"

## 常见问题

### Q: 访问 http://localhost:8080/h2-console 显示 404？

A: 检查：
1. 应用是否正在运行
2. `application.yml` 中是否启用了 H2 控制台：
   ```yaml
   spring:
     h2:
       console:
         enabled: true
         path: /h2-console
   ```

### Q: 连接失败，提示 "Database not found"？

A: 检查：
1. JDBC URL 是否正确：`jdbc:h2:file:./data/essayscoring`
2. 数据库文件是否存在：`data/essayscoring.mv.db`
3. 如果文件不存在，先启动一次应用，让 Hibernate 自动创建表

### Q: 如何重置数据库？

A: 
1. 停止应用
2. 删除 `data/essayscoring.mv.db` 文件
3. 重新启动应用，表结构会自动创建

### Q: 如何备份数据库？

A: 
1. 停止应用
2. 复制 `data/essayscoring.mv.db` 文件到其他位置
3. 或者使用 H2 控制台执行：
   ```sql
   SCRIPT TO 'backup.sql';
   ```

### Q: 如何导入 SQL 脚本？

A: 在 H2 控制台中：
1. 点击 "Run Script" 按钮
2. 选择 SQL 文件
3. 执行脚本
