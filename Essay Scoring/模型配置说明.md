# 如何修改调用的AI模型

## 📋 模型配置位置

模型配置在 `src/main/resources/application.yml` 文件中。

## 🔧 修改方法

### 方法一：直接修改配置文件（推荐）

编辑 `src/main/resources/application.yml` 文件，找到以下部分：

```yaml
spring:
  cloud:
    ai:
      deepseek:
        api-key: ${DEEPSEEK_API_KEY:your-api-key-here}
        chat:
          options:
            model: deepseek-chat  # 👈 在这里修改模型名称
            temperature: 0.3      # 👈 温度参数（0-1，越低越稳定）
            max-tokens: 2000      # 👈 最大输出token数
```

### 方法二：使用环境变量

也可以通过环境变量设置：

**Windows:**
```cmd
set DEEPSEEK_MODEL=deepseek-chat
set DEEPSEEK_TEMPERATURE=0.3
set DEEPSEEK_MAX_TOKENS=2000
```

**Linux/Mac:**
```bash
export DEEPSEEK_MODEL=deepseek-chat
export DEEPSEEK_TEMPERATURE=0.3
export DEEPSEEK_MAX_TOKENS=2000
```

然后在 `application.yml` 中使用：
```yaml
spring:
  cloud:
    ai:
      deepseek:
        chat:
          options:
            model: ${DEEPSEEK_MODEL:deepseek-chat}
            temperature: ${DEEPSEEK_TEMPERATURE:0.3}
            max-tokens: ${DEEPSEEK_MAX_TOKENS:2000}
```

## 🤖 可用的模型列表

### DeepSeek 系列

| 模型名称 | 说明 | 适用场景 |
|---------|------|---------|
| `deepseek-chat` | DeepSeek-V3 对话模型 | 通用评分，平衡性能与成本 |
| `deepseek-reasoner` | DeepSeek-R1 推理模型 | 需要深度推理的评分场景 |

> **推荐**：作文评分场景使用 `deepseek-chat`，性价比最高。如需更细致的推理和反馈，可切换 `deepseek-reasoner`。

> **注意**：模型名称可能会更新，请以 [DeepSeek 官方文档](https://platform.deepseek.com/api-docs/) 为准。

## ⚙️ 参数说明

### model（模型名称）
- **默认值**：`deepseek-chat`
- **说明**：要使用的AI模型名称
- **示例**：`deepseek-chat`、`deepseek-reasoner`

### temperature（温度参数）
- **默认值**：`0.3`
- **范围**：0.0 - 2.0
- **说明**：
  - 值越低（如 0.1-0.3）：输出更稳定、更确定，适合评分任务
  - 值越高（如 0.7-1.0）：输出更随机、更有创意
- **推荐值**：评分任务建议使用 `0.3` 或更低

### max-tokens（最大输出长度）
- **默认值**：`2000`
- **说明**：AI返回内容的最大token数
- **建议**：
  - 评分结果：2000 足够
  - 详细反馈：可以增加到 3000-4000

## 📝 配置示例

### 示例1：使用默认模型（适合日常评分）
```yaml
spring:
  cloud:
    ai:
      deepseek:
        chat:
          options:
            model: deepseek-chat
            temperature: 0.3
            max-tokens: 2000
```

### 示例2：使用推理模型（适合高质量反馈）
```yaml
spring:
  cloud:
    ai:
      deepseek:
        chat:
          options:
            model: deepseek-reasoner
            temperature: 0.2
            max-tokens: 4000
```

## 🔄 修改后如何生效

1. **修改配置文件**后，需要**重启应用**才能生效
2. 重启命令：
   ```bash
   # 停止当前运行的应用（Ctrl+C）
   # 然后重新启动
   mvn spring-boot:run
   ```

## 💡 模型选择建议

### 根据使用场景选择：

1. **开发/测试环境**
   - 推荐：`deepseek-chat`
   - 原因：速度快、成本低

2. **生产环境（追求质量）**
   - 推荐：`deepseek-chat`
   - 原因：DeepSeek-V3 评分准确、中文能力强

3. **深度分析场景**
   - 推荐：`deepseek-reasoner`
   - 原因：推理能力更强，反馈更细致

---

## ⚠️ 注意事项

1. **API Key权限**：确保您的API Key有权限使用所选模型
2. **成本差异**：`deepseek-reasoner` 推理成本略高于 `deepseek-chat`
3. **响应速度**：`deepseek-reasoner` 响应可能稍慢（深度推理）
4. **模型可用性**：如模型不可用，请查看 [DeepSeek 官方文档](https://platform.deepseek.com/api-docs/)
