# 快速启动指南

## 问题：数据库不存在

如果遇到 `Unknown database 'essayscoring'` 错误，请按以下步骤操作：

---

## 方案一：创建 MySQL 数据库（推荐）

### 步骤 1：登录 MySQL

**方式一：命令行**
```bash
mysql -u root -p
```
然后输入密码（根据你的 `application.yml` 配置，默认是 `123456789`）

**方式二：使用 MySQL Workbench 或其他图形化工具**

### 步骤 2：执行创建数据库命令

在 MySQL 命令行或工具中执行：

```sql
CREATE DATABASE IF NOT EXISTS essayscoring 
CHARACTER SET utf8mb4 
COLLATE utf8mb4_unicode_ci;
```

或者直接执行项目根目录下的 `create_database.sql` 文件。

### 步骤 3：验证数据库创建

```sql
SHOW DATABASES LIKE 'essayscoring';
```

如果看到 `essayscoring` 数据库，说明创建成功。

### 步骤 4：重启应用

```bash
mvn spring-boot:run
```

---

## 方案二：临时切换到 H2 数据库（快速测试）

如果你想快速测试应用，可以临时切换到 H2 数据库：

### 步骤 1：修改 `src/main/resources/application.yml`

将 MySQL 配置注释掉，启用 H2 配置：

```yaml
spring:
  application:
    name: essay-scoring-system
  
  # 临时使用 H2 数据库（文件数据库，数据会保存到 data/essayscoring.mv.db）
  datasource:
    url: jdbc:h2:file:./data/essayscoring
    driver-class-name: org.h2.Driver
    username: sa
    password: 
  
  # JPA配置
  jpa:
    hibernate:
      ddl-auto: update
    show-sql: true
    properties:
      hibernate:
        dialect: org.hibernate.dialect.H2Dialect
        format_sql: true
  
  # 启用 H2 控制台（可选）
  h2:
    console:
      enabled: true
      path: /h2-console
  
  # 注释掉 MySQL 配置
  # datasource:
  #   url: jdbc:mysql://localhost:3306/essayscoring?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
  #   driver-class-name: com.mysql.cj.jdbc.Driver
  #   username: root
  #   password: 123456789
```

### 步骤 2：创建数据目录（可选）

```bash
mkdir data
```

应用会自动创建数据库文件。

### 步骤 3：重启应用

```bash
mvn spring-boot:run
```

### 步骤 4：访问 H2 控制台（可选）

访问：http://localhost:8080/h2-console

连接信息：
- JDBC URL: `jdbc:h2:file:./data/essayscoring`
- 用户名: `sa`
- 密码: （留空）

---

## 推荐方案

- **开发/测试环境**：使用 H2 数据库（方案二），无需安装 MySQL
- **生产环境**：使用 MySQL 数据库（方案一），数据更稳定

## 常见问题

### Q: MySQL 连接失败怎么办？

A: 检查：
1. MySQL 服务是否启动
2. 用户名和密码是否正确（在 `application.yml` 中配置）
3. 端口号是否正确（默认 3306）

### Q: H2 数据库文件在哪里？

A: 文件保存在项目根目录下的 `data/essayscoring.mv.db`

### Q: 如何切换回 MySQL？

A: 修改 `application.yml`，注释掉 H2 配置，启用 MySQL 配置，并确保已创建数据库。
